Schedule
Configure automated daily runs on your VPS
Cron Setup (Recommended)

Open crontab with crontab -e and add one of the lines below:

DAILY AT 7:00 AM (RECOMMENDED)
# Run GA4 agent every day at 7:00 AM Berlin time
0 7 * * * cd /var/www/ga4-agent/backend && venv/bin/python agent.py >> logs/cron.log 2>&1
MONDAY–FRIDAY ONLY
# Weekdays only
0 7 * * 1-5 cd /var/www/ga4-agent/backend && venv/bin/python agent.py >> logs/cron.log 2>&1
Cron reference0 7 * * * → 7:00 AM every day
0 7 * * 1-5 → 7:00 AM Mon–Fri
*/30 * * * * → every 30 minutes
0 */6 * * * → every 6 hours
Set server timezone to Germany:
timedatectl set-timezone Europe/Berlin
ENDOFFILE
